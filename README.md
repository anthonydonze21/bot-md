# CodexBot
iTicket make with Discord.JS **v13.6**

### For install this
```npm install```
### For run this
```node iTicket.js```


# Installer le bot discord sous linux

1. - Installer Node (dernière version)
`sudo apt install nodejs@latest`

2. - Installer PM2 avec npm
`npm i pm2`

3. - Aller dans le dossier du bot discord
`cd Example/iTicket`

Lancer le bot (être dans le dossier du bot discord)
`pm2 start iTicket.js`

Redémarrer le bot (être dans le dossier du bot discord)
`pm2 restart iTicket.js`

Voir les logs du bot discord (ouvrir ce moniteur avant le lancement du bot discord)
`pm2 monit`
